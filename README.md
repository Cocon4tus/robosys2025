四則演算コマンド

標準入力から数字、演算子を読み込み、計算した結果を出力するコマンドです。

前提環境

・ python
・ テスト済みバージョン：3.7~3.10
・ Ubuntu 22.04.5

インストール方法

・ 下記のコマンドを入力すると使用できるようになります。\
  	
	- $ git clone https://github.com/Cocon4tus/robosys2025
	- $ cd robosys2025
	

実行

・ 変数が二つ、演算子を一つ入力することで自動で計算結果を見ることができます。\
・ 演算子について対応しているのは'+' '-' '*' '/'　の四つのみであり、他の文字を入力すると'error'と表示されるようになっています。\
	
・実行例
	
	- $ echo "3+5" | ./sisokuenzan
	- $ 8
